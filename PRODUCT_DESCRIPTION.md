## Project Purpose
**"Taking Flight to 2026"** is a voice-first marketing and strategic automation platform designed for the KSU Athletics Executive Team. It serves as a real-time "Command Cockpit" for the Power Four ascent, ensuring that the $10M revenue pipeline and institutional system strengths are tracked, managed, and executed with precision.

## Core Capabilities
- **Voice-First Orchestration**: Gemini 2.5 Live API enables real-time, low-latency voice control of the strategic dashboard.
- **Cognitive Knowledge Base**: A "Second Brain" containing the full strategic playbook, mission parameters, and revenue targets.
- **Cloud Persistence**: Full Supabase integration for multi-user synchronization and data durability.
- **Strategic Visualization**: Real-time metrics tracking for Revenue Gaps, Pipeline Value, and Operational Efficiency.

## Technical Architecture
- **Frontend**: React 19, Tailwind CSS (Executive High-Contrast Theme).
- **Intelligence**: Google Gemini 2.5 Flash & Pro (Live & Text modalities).
- **Persistence**: Supabase (PostgreSQL) + LocalStorage Fallback.
- **Components**: Lucide Icons, Custom Audio Processing for low-latency voice capture.
